Semántica

Nuevos elementos semánticos para cabeceras, encabezados, piés de página, secciones, artículos, menús de navegación, contenido secundario, figuras y nuevos atributos y elementos de formulario.

Ver ejemplo de Semántica

Etiquetas Semánticas

<!DOCTYPE html> -> Es el DOCTYPE para HTML5, más simplificado que los de XHTML 1.0, y permite utilizar todas las capacidades del nuevo estándar.

<header></header> -> Representa la cabecera del documento.

<hgroup></hgroup> -> Nos permite tener múltiples encabezados h1...h6, sin afectar el SEO del documento.

<footer></footer> -> Representa el pié de página del documento.

<section></section> -> Representa un área o zona de contenido específico.

<article></article> -> Representa un contenido independiente.

<nav></nav> -> Representa un menú de navegación en el documento.

<aside></aside> -> Representa un contenido secundario y/o sin relevancia en el documento.

<figure> -> Representa una figura en el documento.
    <figcaption></figcaption> -> Representa la nota al pié de la figura.
</figure>
